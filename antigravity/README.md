# NovaMedia Studio — Antigravity Workspace

## Purpose
This folder is for Google Stitch UI prototyping.

## Workflow
1. Open STITCH_PROMPT.md → copy to labs.google/fx/tools/stitch
2. Stitch generates HTML prototype
3. Save output here as stitch-output.html
4. Integrate the best elements into the Next.js site
